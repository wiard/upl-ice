# demo_WARP_01_agent_handoff_payment_gate

This demo shows that an agent handoff and a mock payment release can be gated by an explicit warrant status under a bounded local policy.

Run:

```bash
python3 run.py
```

Outputs:

- `witnesses/mandate.json`
- `witnesses/route.json`
- `witnesses/quote.json`
- `witnesses/contract.json`
- `witnesses/check.json`
- `witnesses/ruleout.json`
- `results/summary.json`
- `results/trace.log`
- `results/judgment.json`
- `results/mock_ledger.json`
